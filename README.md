# Clear Maker 2A (生徒用アプリ)

**AIを活用した「生徒向け」自動解答添削システム（スプレッドシート連動版）**

Clear Maker 2A は、生徒が自分のスマホやタブレットから解答写真を撮影してアップロードするだけで、即座にAIの添削結果を受け取ることができるWebアプリケーションです。
裏側で動作するGoogle Apps Script（GAS）とGoogleスプレッドシートが連動することで、問題文や模範解答の入力を生徒側で行う必要がなく、高速かつ正確な添削を実現します。

---

## 🌟 主な機能

*   **スプレッドシート完全連動プルダウン**:
    *   先生がスプレッドシートに問題を登録すると、自動的に生徒画面のプルダウンメニューに最新の問題一覧が反映されます。
*   **爆速・超軽量フィードバック**:
    *   処理を軽くするため、AIからの返答を「正誤」「合格/再チャレンジの判定」「20文字程度の一言コメント」の3点のみに削ぎ落とし、生徒を待たせないサクサクな動作を提供します。
*   **今日のまとめ表示**:
    *   添削後にその日の提出履歴を分析し、レベルアップ判定の場合でも誤答が含まれていれば、その原因とテーマを示したうえで、解き直すべき類題を別スプレッドシートから引用して表示します。
*   **生徒番号の自動記憶**:
    *   初回利用時に生徒番号（ID）を一度登録すればブラウザに記憶され、毎回の入力をスキップできます。
*   **安全なアーキテクチャ**:
    *   GeminiのAPIキーはすべてサーバー側（GAS）に隠蔽されており、生徒側の画面からAPIキーが漏洩する心配はありません。

## ⚙️ システム構成

このシステムは以下の3つの要素から成り立っています。

1.  **Frontend (生徒が操作する画面)**
    *   `index.html` / `student.js` / `student.css`
    *   ブラウザやPWAとして配布し、生徒が写真をアップロードする画面です。
2.  **Backend (GAS: 先生が管理するサーバー)**
    *   `backend_GAS_spreadsheet.gs`
    *   フロントエンドからの画像を受け取り、スプレッドシートからテキストを抽出し、AIへ指示を出します。
3.  **Database (Googleスプレッドシート)**
    *   「第1回 大問1」などの問題リストと、問題・模範解答のテキスト（TeXコード）を一元管理します。

## 🚀 将来の改善計画 (Planned Improvements)

利用規模の拡大に伴い、必要に応じて以下の機能を順次追加する予定です。

*   **同時アクセス時のエラー対策（排他制御と自動リトライ）**:
    *   **背景**: 多数の生徒が同時に「採点」ボタンを押すと、スプレッドシートへの保存時に書き込み競合（Lock timeout等）が発生する可能性があります。
    *   **対策**:
        1.  **バックエンド (GAS)**: `LockService` を導入し、スプレッドシートへの書き出し処理を一時的に保護・順次実行させるように改善します。
        2.  **フロントエンド (JS)**: 通信失敗時に指数バックオフ（徐々に待ち時間を増やして再試行する仕組み）を用いた自動リトライ機能を搭載します。
    *   **状況**: 現時点では「0〜5秒のランダム待機」による負荷分散で対応しており、今後の利用状況を見て導入を判断します。
