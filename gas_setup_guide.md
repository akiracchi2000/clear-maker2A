# Clear Maker 2A: GAS・スプレッドシート連携セットアップ手順

生徒用アプリ「Clear Maker 2A」を、GoogleスプレッドシートおよびGoogle Apps Script（GAS）と連携させるための全手順です。

---

## 1. スプレッドシート（問題リスト）の準備

まず、AIへ読み込ませる「問題と解答」を管理するスプレッドシートを作ります。

1. 新しいGoogleスプレッドシートを作成します。最初のシート（一番左のタブ）が**「問題リスト」**として読み込まれます。シート名は「数学」など自由に変更していただいて構いません。
2. 1行目に各列の見出し（ヘッダー）を作り、**2行目以降**に以下のルールでデータを入力します。

| A列 | B列 | C列 | D列 | E列 |
| :--- | :--- | :--- | :--- | :--- |
| **問題ID** | **回・単元** | **表示ラベル** | **問題文(TeX等)** | **模範解答(TeX等)** |
| `M-001` | 第1回 | 大問1 | `x^2 - 4x + 4 = 0 を解け` | `(x-2)^2 = 0 より、x = 2` |
| `M-002` | 第1回 | 大問2 | `$x>0$ のとき…` | `...` |

*(※B列がアプリ1つ目のプルダウン、C列が2つ目のプルダウンに表示されます。A列のシステムIDは他と被らないようにしてください)*

4. **【重要】成績管理用シートの追加**
   生徒が提出した結果を自動保存するための新しいシートを追加します。
   スプレッドシートの左下の「＋」ボタンを押してシートを追加し、名前を必ず **`提出履歴`** に変更してください。
5. 「提出履歴」シートの1行目に、以下の見出しを作成しておきます。
   * A列: `日時`
   * B列: `生徒番号`
   * C列: `問題ID`
   * D列: `判定`
   * E列: `詳細コメント`
   * F列: `生テキスト`

6. スプレッドシートのURLを確認します。
   `https://docs.google.com/spreadsheets/d/コレ/edit`
   の「コレ」にあたる文字列をコピーして控えておきます。これが **スプレッドシートID** です。

---

## 2. Google Apps Script (GAS) の設定

1. ブラウザで [Google Apps Script](https://script.google.com/) を開き、「新しいプロジェクト」を作成します。
2. もともと書かれている `function myFunction() { ... }` をすべて消去します。
3. ローカルにある **`backend_GAS_spreadsheet.gs`** のソースコードを全選択してコピーし、GASのエディタに貼り付けます。
4. 貼り付けたコードの中で、以下の2つの文字列を書き換えます。

```javascript
// 45行目付近: ご自身のGemini APIキーにする
const STUDENT_APP_API_KEY = "ここにAPIキーを入力してください";

// 66行目付近: 手順1でコピーしたスプレッドシートIDにする
const SPREADSHEET_ID = "YOUR_SPREADSHEET_ID_HERE";
```
5. 「プロジェクトの保存（フロッピーアイコン）」をクリックします。

---

## 3. デプロイとURLの取得

1. 画面右上の青い **「デプロイ」** ボタン ＞ **「新しいデプロイ」** を選びます。
2. 左側の歯車(⚙️)アイコンをクリックし、**「ウェブアプリ」** をチェックします。
3. **【重要】**「アクセスできるユーザー」を「自分のみ」から **「全員」** に変更します。
4. そのまま「デプロイ」をクリックします。
5. （※権限の承認画面が出た場合は、自分のアカウントを選択 ➔ 左下の「詳細」 ➔ 「〇〇（安全ではないページ）に移動」 ➔ 「許可」の順番に進めます）
6. 最終画面で **ウェブアプリのURL** （`https://script.google.com/.../exec`）が表示されるので、コピーします。

---

## 4. アプリ（student.js）との連携

1. パソコンの作業フォルダにある **`student.js`** をエディタやメモ帳で開きます。
2. 一番上のほうにある `GAS_API_URL` の中身を、手順3でコピーした**新しいウェブアプリのURL**に貼り替えます。

```javascript
// student.js の3行目付近
const GAS_API_URL = 'https://script.google.com/macros/s/あなたの新しいURL/exec';
```

3. `student.js` を保存し、ブラウザで `index.html` を開きます。
4. プルダウンの項目が「-- データ読込中... --」から一瞬で切り替わり、スプレッドシートの内容（B列・C列）が表示されれば**連携成功**です！
